# Voice Command AI API
This API receives voice commands, processes them using AI, and returns appropriate responses or actions based on the commands given.

## Features
- Receive voice commands via HTTP POST requests.
- Process commands using AI to understand intent.
- Return responses or trigger actions based on processed commands.

The AI has access to tools including:
- Look up a patient by name or id
- Schedule an appointment
- Cancel an appointment
- Retrieve patient medical history
- Provide medication information
- Answer general health-related questions
